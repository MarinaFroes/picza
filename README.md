## Introduction

- This mini pizza pic gallery has been created using **webpack scaffold** based on the references linked bellow. My goal was to learn how to use Webpack 4.

## Tech stack

- HTML
- SASS
- JQuery
- Webpack

## How to use it

To bundle this application and start the server, you can run:

```
$ npm run app:dev
```
